Инструкция развертывания

Точная последовательность действий для мгновенного обхода

1. Загрузка
zapret-discord-youtube-1.9.9a.zip
Архив содержит исполняемые модули и скрипты.
2. Распаковка
C:\zapret\
Рекомендуется разместить в корне диска.
3. Запуск
Discord Zapret.exe
Обязательно выполнить от имени администратора.
>> После запуска Discord Zapret.exe фильтрация активируется мгновенно. Для деинсталляции используйте remove_service.bat (администратор).
